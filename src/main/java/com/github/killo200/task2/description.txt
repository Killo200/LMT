Вася и Петя купили коробку с конфетами. Чтобы разделить их между собой, они решили поиграть
в придуманную ими игру. Игроки поочередно берут некоторое положительное количество
конфет, но не более заранее определенного для всей игры числа. Взятые конфеты игрок кладет
рядом с собой. Тот игрок, ход которого опустошает коробку, съедает свои конфеты, а
проигравший кладет свои конфеты обратно в коробку. После этого игра продолжается, при этом
первый ход делает проигравший. Целью игры является съесть как можно больше конфет. Сколько
конфет может съесть Вася, начинающий игру, если обе стороны придерживаются оптимальной
для себя стратегии?

Входные данные:
Единственная строка входного файла содержит в себе два целых числа n и m, разделенных
пробелами. n — число конфет в коробке во время начала игры, m — максимальное количество
конфет, которое может быть взято одним игроком за ход.

Выходные данные:
Должно быть напечатано одно целое число, равное количеству конфет, которое может съесть
Вася.
Ограничения 1 ≤ m ≤ n ≤ 100

Пример:
Входные данные

5 2

Выходные данные

3
