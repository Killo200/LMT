Формат входного файла:
Первая строка файла содержит грузоподъемность вертолета. Остальные строки представляют
собой описание необходимых предметов: сначала записано требуемое количество, потом символ
х, название, масса и полезность одного предмета.
Формат выходного файла:
Необходимо вывести список предметов в алфавитном порядке названий в формате количество х
название.

Пример:
Входные данные
100
5 x M16 5 40
5 x AmmoBox 10 100
2 x GreenadeBox 15 80


Выходные данные

5 x AmmoBox
1 x GreenadeBox
5 x M16
